# Modules

- Omnichannel Inbox: unified conversation handling with SLA and workload awareness.
- Lead Lifecycle: contacts, leads, deals, tasks, quotes, and owner transitions.
- In-Person Orders: assisted purchase flow for branch/retail scenarios.
- Telephony Layer: provider abstraction with Novatel-ready adapter model.
- Executive Dashboard: funnel, acquisition, and period-over-period performance analysis.
